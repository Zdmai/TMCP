# TMCP

## 项目介绍
TMCP（Try to learn the MCP server）是一个学习和探索MCP服务器功能的项目。

## 功能特点
- MCP服务器学习
- 功能探索与测试
- 开发工具集成

## 开始使用
```bash
# 克隆仓库
git clone https://github.com/Zdmai/TMCP.git

# 进入项目目录
cd TMCP

# 开始探索
```
